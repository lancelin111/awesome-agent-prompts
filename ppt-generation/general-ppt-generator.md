# 通用PPT生成器 Prompt

## Prompt

```
你是一个专业的PPT内容设计专家。请根据以下要求生成PPT内容：

【任务】
帮我创建一个关于[主题]的PPT大纲和详细内容。

【要求】
1. PPT总页数：[X]页
2. 目标受众：[描述受众特征]
3. 演讲时长：[X]分钟
4. 风格要求：[正式/轻松/学术/商务]

【输出格式】
请按以下格式输出：

# PPT标题

## 第1页 - 封面
- 标题：
- 副标题：
- 演讲者：
- 日期：

## 第2页 - 目录
- 列出主要章节

## 第3-N页 - 内容页
对于每一页，请提供：
- 页面标题
- 核心观点（3-5个要点）
- 演讲者备注（简要说明要点）
- 建议的视觉元素（图表/图片类型）

## 最后一页 - 总结/致谢
- 关键要点回顾
- 联系方式
- Q&A

【注意事项】
- 每页内容要简洁，避免文字过多
- 使用项目符号而非长段落
- 确保逻辑流畅，过渡自然
- 考虑视觉效果和可读性
```

## 使用示例

### 输入
```
主题：人工智能在教育中的应用
PPT总页数：10页
目标受众：高中教师
演讲时长：15分钟
风格要求：轻松但专业
```

### 输出示例
```
# 人工智能重塑教育：机遇与挑战

## 第1页 - 封面
- 标题：人工智能重塑教育
- 副标题：探索AI在课堂中的创新应用
- 演讲者：[姓名]
- 日期：2024年

## 第2页 - 目录
1. AI教育简介
2. 当前应用案例
3. 教学效果提升
4. 实施策略
5. 未来展望

[继续...]
```
