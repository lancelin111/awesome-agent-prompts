```

【数据描述】

- 数据规模：[行数 x 列数]

1. **数据概览**
   - 数据类型分布

2. **单变量分析**
   - 数值型变量分布

4. **数据质量报告**
   - 数据完整性
   - 数据一致性

【输出格式】
- 使用Python代码示例

- 提供业务洞察
