# 函数生成器 Prompt

## 通用函数生成

```
请根据以下要求生成一个函数：

【函数需求】
- 功能描述：[详细描述函数的功能]
- 编程语言：[语言]
- 函数名：[建议的函数名]
- 输入参数：[参数列表和类型]
- 返回值：[返回值类型和含义]

【约束条件】
- 时间复杂度要求：[如 O(n)]
- 空间复杂度要求：[如 O(1)]
- 其他限制：[如不能使用某些库]

【测试用例】
输入：[示例输入]
输出：[期望输出]

【输出要求】
1. 完整的函数实现
2. 详细的注释
3. 错误处理
4. 单元测试代码
5. 使用示例
```

## 算法实现 Prompt

```
请实现[算法名称]算法：

【算法要求】
- 编程语言：[语言]
- 输入格式：[描述]
- 输出格式：[描述]
- 算法说明：[简要说明算法原理]

【实现要求】
1. 使用清晰的变量命名
2. 添加步骤注释
3. 处理边界情况
4. 提供时间和空间复杂度分析

【额外要求】
- 提供优化版本（如果可能）
- 比较不同实现方法
- 提供可视化输出（如果适用）
```

## 设计模式实现

```
请使用[设计模式名称]模式实现以下功能：

【场景描述】
[描述具体的业务场景]

【类结构要求】
- 主要类：[列出需要的类]
- 接口：[列出需要的接口]
- 关系：[描述类之间的关系]

【实现要求】
1. 严格遵循设计模式
2. 提供UML类图（文字描述）
3. 包含使用示例
4. 说明模式的优势
5. 讨论可能的扩展
```
