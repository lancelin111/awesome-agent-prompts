# 代码审查 Prompt

## 综合代码审查

```
请对以下代码进行全面的代码审查：

【代码】
```[编程语言]
[粘贴代码]
```

【审查要求】
请从以下方面进行评估：

1. **代码质量**
   - 可读性和可维护性
   - 命名规范
   - 代码结构
   - 注释质量

2. **功能正确性**
   - 逻辑错误
   - 边界条件处理
   - 异常处理

3. **性能优化**
   - 时间复杂度
   - 空间复杂度
   - 潜在的性能瓶颈

4. **安全性**
   - 安全漏洞
   - 输入验证
   - 敏感信息处理

5. **最佳实践**
   - 设计模式应用
   - SOLID原则
   - DRY原则

【输出格式】
请按以下格式输出审查结果：

## 总体评分：X/10

## 优点
- 列出代码的优点

## 问题与建议

### 严重问题（必须修复）
1. 问题描述
   - 影响：说明影响
   - 建议：修复方案
   - 代码示例：```提供修复后的代码```

### 中等问题（建议修复）
1. ...

### 轻微问题（可选优化）
1. ...

## 重构建议
提供整体的重构方案（如果需要）
```
