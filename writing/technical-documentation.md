# 技术文档写作 Prompt

## API文档生成

```
请为以下代码生成专业的API文档：

【代码】
```[语言]
[粘贴API代码]
```

【文档要求】
1. 概述
   - 功能描述
   - 使用场景
   - 版本信息

2. 接口详情
   - 请求方法
   - 请求URL
   - 请求参数
   - 响应格式
   - 错误码

3. 示例
   - 请求示例
   - 响应示例
   - 常见用例

4. 注意事项
   - 认证方式
   - 限流策略
   - 最佳实践

【输出格式】
使用Markdown格式，包含代码高亮
```

## README文档生成

```
请为我的项目生成一个专业的README文档：

【项目信息】
- 项目名称：[名称]
- 项目类型：[库/框架/应用]
- 主要功能：[描述]
- 技术栈：[列出技术]
- 目标用户：[描述]

【README结构】
1. 项目标题和描述
2. 特性列表
3. 安装指南
4. 快速开始
5. 使用示例
6. API参考
7. 配置说明
8. 贡献指南
9. 许可证
10. 联系方式

【要求】
- 添加徽章（badges）
- 包含截图或演示GIF
- 提供清晰的代码示例
- 使用表格组织信息
```
