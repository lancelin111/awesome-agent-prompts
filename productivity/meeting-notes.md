# 会议纪要 Prompt

## 会议纪要生成器

```
请根据以下会议信息生成专业的会议纪要：

【会议基本信息】
- 会议主题：[主题]
- 会议时间：[日期时间]
- 会议地点：[地点/线上]
- 参会人员：[列表]
- 会议主持：[姓名]

【会议内容】
[粘贴会议记录或要点]

【输出要求】
请生成包含以下内容的会议纪要：

1. **会议概要**
   - 会议目的
   - 主要议题
   - 关键结论

2. **讨论要点**
   - 按议题分类
   - 记录各方观点
   - 标注重要决策

3. **行动计划**
   - 任务项（What）
   - 负责人（Who）
   - 截止时间（When）
   - 完成标准（How）

4. **后续安排**
   - 下次会议时间
   - 需准备的材料
   - 待解决问题

【格式要求】
- 使用项目符号和编号
- 突出重点内容
- 保持简洁清晰
- 添加时间戳（如需要）
```

## 会议总结提炼

```
请将以下冗长的会议记录提炼成简洁的执行要点：

【原始记录】
[粘贴会议记录]

【提炼要求】
1. 提取3-5个核心决策
2. 列出具体行动项
3. 标注风险和依赖
4. 生成一句话总结
5. 制作待办事项清单
```
