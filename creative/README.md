## 创意技法

### SCAMPER法
- **S**ubstitute: 替换
- **C**ombine: 组合
- **A**dapt: 改编
- **M**odify: 修改
- **P**ut to another use: 其他用途
- **E**liminate: 消除
- **R**earrange: 重组

- ⚪ 白帽：事实和数据

- ⚫ 黑帽：批判和谨慎
- 🟡 黄帽：乐观和积极
