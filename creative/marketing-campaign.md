# 营销策划 Prompt

## 营销活动创意生成

```
请为以下产品/服务生成创意营销方案：

【产品信息】
- 产品名称：[名称]
- 产品特点：[列出3-5个特点]
- 目标受众：[描述用户画像]
- 竞争对手：[主要竞品]
- 预算范围：[金额]

【创意要求】
请提供5个不同风格的营销创意：

1. **病毒式传播创意**
   - 核心概念
   - 传播机制
   - 预期效果
   - 执行要点

2. **情感共鸣创意**
   - 情感诉求点
   - 故事线
   - 视觉呈现
   - 用户参与方式

3. **互动体验创意**
   - 互动形式
   - 技术实现
   - 用户激励
   - 数据收集

4. **联名合作创意**
   - 合作对象建议
   - 合作形式
   - 双方利益点
   - 推广计划

5. **社会化营销创意**
   - 社会议题
   - 品牌立场
   - 执行方案
   - 风险控制

【评估维度】
- 创新性（1-10分）
- 可行性（1-10分）
- 成本效益（1-10分）
- 风险等级（低/中/高）
```

## Slogan创作

```
请为品牌创作slogan：

【品牌信息】
- 品牌名称：[名称]
- 品牌定位：[描述]
- 核心价值：[列出]
- 品牌个性：[描述]
- 竞品slogan：[列出参考]

【创作要求】
1. 提供10个不同角度的slogan
2. 每个slogan包含：
   - 中文版本
   - 英文版本（如需要）
   - 创意说明
   - 适用场景

【评判标准】
- 简洁有力
- 易于记忆
- 体现差异化
- 情感连接
- 行动号召力
```

## 内容营销策略

```
请制定内容营销策略：

【业务背景】
- 行业领域：[领域]
- 目标用户：[画像]
- 营销目标：[具体目标]
- 内容预算：[预算]
- 时间周期：[周期]

【策略输出】

1. **内容主题矩阵**
   - 教育类内容（40%）
   - 娱乐类内容（30%）
   - 促销类内容（20%）
   - 品牌类内容（10%）

2. **内容形式规划**
   - 图文内容
   - 视频内容
   - 直播内容
   - 互动内容

3. **发布渠道策略**
   - 自有媒体
   - 社交媒体
   - 合作媒体
   - 付费推广

4. **内容日历**
   - 月度主题
   - 周度规划
   - 热点借势
   - 节日营销

5. **效果评估指标**
   - 曝光量
   - 互动率
   - 转化率
   - ROI
```
